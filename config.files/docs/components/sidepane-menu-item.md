## What this component is for

`SidepaneMenuItem` is the 64×64px clickable cell that makes up the Zoho Analytics left-side navigation panel. Each item displays an icon slot above a short text label and switches between three visual states to communicate user interaction.

**Use cases:**
- Rendering all primary navigation destinations (Explorer, Dashboards, Reports, Data, Ask Zia, Data Sources, Code Studio, Auto ML, Settings, Trash)
- Indicating the currently active section (Selected state — blue fill, white icon + label)
- Providing hover feedback as users move between nav items
